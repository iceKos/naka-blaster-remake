## Requirements
* install [node](https://nodejs.org/es/download/). </br>
* install [git](https://git-scm.com/downloads).
## Download & install
```console
git clone https://github.com/Koutawars/bomber-demo.git
cd bomber-demo
npm install
```
## Run
```console
node server/app.js
```
